给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

**输入：**

```
head = [1,2,3,4,5], n = 2

```

**输出：**

```
[1,2,3,5]


```

```
int cur = 0; // 从最后计数。
// 这个是用不断的递归的方式到达底部
ListNode *removeNthFromEnd(ListNode *head, int n)
{
    if (!head)
        return NULL;
    head->next = removeNthFromEnd(head->next, n);
    cur++;
    if (n == cur)
        return head->next;
    return head;
}
```
