我用的是element plus ,主要代码如下，显示路由的代码
```js
const router_config = [
  {
    path: '/',
    name: 'login',
    component: LoginView,
  },
  {
    path: '/home',
    name: 'home',
    component: HomeView,
    children: [
      {
        path: '/contre',
        name: 'contre',
        component: MonitorContreView,
        alias: '' /** // 将父路由路径作为这个的别名，意味着默认显示这个。 */,

        component: SettingView,
```

然后是模板的代码,我这里将key设置成name，期望没有重名的。
```vue
<template>
  <div>
    <el-container>
      <el-aside width="200px">
        <el-menu router>
          <template v-for="item in menus" :key="item.name">
            <el-sub-menu v-if="item.children" :index="item.path">
              <template #title>
                <el-icon>
                  <component :is="item.icon" />
                </el-icon>
                <span>{{ item.name }}</span>
              </template>
              <el-menu-item v-for="child in item.children" :key="child.name" :index="child.path">
                {{ child.name }}
              </el-menu-item>
            </el-sub-menu>
            <el-menu-item v-else :index="item.path">
              <!-- <el-icon>
              <component :is="item.icon" />
            </el-icon> -->
              <template #title>{{ item.name }}</template>
            </el-menu-item>
          </template>
        </el-menu>
      </el-aside>
      <!-- 上边的菜单el-menu router，表示可以用这个路由-->
      <el-main><router-view /></el-main>

    </el-container>
  </div>

</template>
```
