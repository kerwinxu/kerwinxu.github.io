qt默认没有安装mysql的驱动的，需要从源码编译，如下是编译步骤。
# 下载qt源码
安装程序里有，不过有些版本没有提供源码，6.8.3提供了源码

# 拷贝
将"C:\Program Files\MySQL\MySQL Server 8.0\lib"文件夹下的"libmysql.dll"和"libmysql.lib"拷贝到编译器目录，我的是"D:\Qt\6.8.3\mingw_64\bin"，

# 编译
打开"D:\Qt\6.8.3\Src\qtbase\src\plugins\sqldrivers\CMakeLists.txt"，这个是一个项目，然后打开qsqlitedriverplugins_cmake_files下的".make.conf"文件，除了第一行的版本号，其他的是我新增的，请注意，不能有空格，所以我是放在qt目录下了，
```
set(QT_REPO_MODULE_VERSION "6.8.3")
SET(FEATURE_sql_mysql ON)
SET(MySQL_INCLUDE_DIR "D:/Qt/6.8.3/Src/qtbase/src/plugins/sqldrivers/mysql2/include")
SET(MySQL_LIBRARY "D:/Qt/6.8.3/Src/qtbase/src/plugins/sqldrivers/mysql2/lib/libmysql.lib")
```
然后构建，最终生成"D:\Qt\6.8.3\Src\qtbase\src\plugins\sqldrivers\build\Desktop_Qt_6_8_3_MinGW_64_bit-Debug\plugins\sqldrivers"目录里的东西，将这些拷贝到"D:\Qt\6.8.3\mingw_64\plugins\sqldrivers"

# 验证
pro文件添加
```
QT += sql
```
代码
```c

#include <QSqlDatabase> // QT数据库

{
```
