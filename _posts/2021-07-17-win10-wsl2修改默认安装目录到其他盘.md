# 1.  查看已安装的linux发行版本

```
wsl -l --all -v
```

# 2.  导出分发版为tar文件到d盘

```
wsl --export Ubuntu-20.04 d:\wsl-ubuntu20.04.tar
```

# 3\. 注销当前分发版

```
wsl --unregister Ubuntu-20.04
```

# 4\. 重新导入并安装WSL在D盘

```
wsl --import Ubuntu-20.04 d:\wsl-ubuntu20.04 d:\wsl-ubuntu20.04.tar --version 2
```

# 5\. 设置默认登陆用户为安装时用户名

```
ubuntu2004 config --default-user USERNAME
```

```

```
