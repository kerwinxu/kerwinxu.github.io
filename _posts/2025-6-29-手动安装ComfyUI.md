# 下载
```
git clone https://github.com/comfyanonymous/ComfyUI.git  --depth 1
```

# 安装支持inter集成显卡的pytorch，可选
修改requirements.txt文件，删除如下3个 torch torchvision torchaudio

# 安装
进入目录后，我这里用3.12版本，官方最推荐，
```
pipenv install --python 3.12
pipenv shell 
pip install -r requirements.txt
```
# 可选安装inter集成显卡
进入虚拟环境，然后pip安装，
```
pipenv shell
pip uninstall torch torchvision torchaudio
pip install torch==2.7.0 torchvision==0.22.0 torchaudio==2.7.0 --index-url https://download.pytorch.org/whl/xpu
pip install intel-extension-for-pytorch==2.7.10+xpu --extra-index-url https://pytorch-extension.intel.com/release-whl/stable/xpu/cn/
```

# 运行

```
pipenv shell
python main.py --cpu --windows-standalone-build
```
或者
```
pipenv run python main.py --cpu --windows-standalone-build
```


注意，inter的显卡跟pytorch不匹配，需要添加参数"--cpu"，不然会报错。

# 安装ComfyUI-Manager

```
pipenv shell
cd custom_nodes
git clone https://github.com/Comfy-Org/ComfyUI-Manager --depth 1
cd ComfyUI-Manager
pip install -r requirements.txt


```


# 概念

## CheckPoint  模型检查点
是真正的Stable Diffusion模型，包括了生成图像所需的所有内容，

## LoRA 模型
小型的补丁文件，用于修改样式，

## AVE 模型
变分自编码器,

## Clip 模型
文字和图像之间相似度的，相当于文本编码器。
